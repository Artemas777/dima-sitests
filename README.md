# Дмитрий — Банковский консультант

Премиальный одностраничный лендинг для консультанта по банковским продуктам.

## Структура проекта

```
dmitry-landing/
├── index.html
├── css/
│   └── style.css
└── js/
    └── main.js
```

## Запуск

Файлы не требуют сборки — это чистый HTML/CSS/JS.

1. Скачайте/склонируйте репозиторий.
2. Откройте `index.html` в браузере — либо запустите локальный сервер:
   ```bash
   npx serve .
   # или
   python3 -m http.server
   ```

## Публикация на GitHub Pages

1. Создайте репозиторий и загрузите содержимое этой папки в его корень.
2. В настройках репозитория → **Pages** выберите ветку `main` и папку `/root`.
3. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

## Что стоит поменять перед публикацией

- Ссылки `href="#"` на кнопках «Написать в Telegram» / «Написать в WhatsApp» — заменить на реальные `https://t.me/...` и `https://wa.me/...`.
- Цифры в блоке статистики (`data-count`) и в блоке «Обо мне» — привести в соответствие с реальными показателями.
- Суммы вознаграждений в разделе «Как я зарабатываю» — актуализировать при изменении условий банков-партнёров.
