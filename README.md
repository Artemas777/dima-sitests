# Дмитрий — лендинг консультанта по банковским продуктам

Премиальный статический лендинг: чистый HTML/CSS/JS без сборщиков и зависимостей
(кроме шрифтов Google Fonts, подключённых через `<link>`).

## Структура проекта

```
.
├── index.html
├── css/
│   ├── main.css          # точка входа, импортирует остальные файлы
│   ├── variables.css      # дизайн-токены: цвета, типографика, отступы, тени
│   ├── base.css           # reset и базовые стили
│   ├── nav.css            # навигация
│   ├── buttons.css        # кнопки
│   ├── hero.css           # hero-блок и 3D-карта
│   ├── about.css          # блок «Обо мне»
│   ├── products.css       # карточки банковских продуктов
│   ├── reward.css         # карточки вознаграждения (стиль ваучера)
│   ├── misc-sections.css  # «Почему я», FAQ, CTA, футер
│   ├── animations.css     # keyframes и scroll-reveal утилиты
│   └── responsive.css     # адаптив и мобильное меню
├── js/
│   └── main.js            # навигация, scroll-reveal, 3D-карта, FAQ-аккордеон
└── assets/                # папка для медиа (пока пустая)
```

## Как запустить

Просто откройте `index.html` в браузере — сборка не требуется.

Для локальной разработки удобно поднять простой сервер (чтобы `@import` в CSS
и относительные пути работали без ограничений `file://`):

```bash
python3 -m http.server 8080
# затем откройте http://localhost:8080
```

## Перед публикацией — замените плейсхолдеры

В коде оставлены явные точки для настройки под реальные данные:

- `index.html` → ссылки `href="#"` с комментарием `<!-- TODO -->` рядом с
  кнопками «Написать в Telegram» / «Написать в WhatsApp» и в футере —
  замените на реальные `https://t.me/...` и `https://wa.me/...`.
- Цифры в hero (`4+ года`, `12 банков-партнёров`, `600+ заявок`) и в блоке
  «Обо мне» — иллюстративные, замените на реальные показатели.
- Суммы вознаграждения в разделе «Вознаграждение» — проверьте актуальные
  условия перед публикацией.

## Технические заметки

- Без фреймворков и сборщиков: ванильные HTML/CSS/JS.
- Анимации уважают `prefers-reduced-motion`.
- Интерактивная карта в hero не требует изображений — реализована на CSS/SVG.
- Все декоративные SVG помечены `aria-hidden`, интерактивные элементы имеют
  видимый фокус (`:focus-visible`) для доступности с клавиатуры.
